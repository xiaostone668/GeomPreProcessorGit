## 快速开始

### 启动程序

运行 `ParasolidGeometryRepair` 可执行文件。

### 基本工作流

1. **导入模型**
   - 文件 → 打开 或 直接拖放文件到窗口
   - 支持的格式：STEP (.stp/.step), IGES (.igs/.iges), Parasolid XT (.x_t/.x_b)

2. **执行检查修复** (推荐首先执行)
   - 几何 → 检查修复
   - 点击"检查"按钮
   - 查看检查结果
   - 选择问题并修复

3. **保存结果**
   - 文件 → 保存 或 文件 → 导出

## 五大功能详解

### 缝合

**用途**: 将多个独立的面缝合为连续的壳体或实体

**使用步骤**:
1. 几何 → 缝合
2. 设置缝合公差 (默认 1e-6)
3. 可选：点击"分析"查看自由边数量
4. 点击"缝合"或"智能缝合"

**参数说明**:
- **公差**: 缝合时允许的最大间隙
- **最大公差**: 自动扩展的最大公差限制
- **允许非流形**: 允许生成非流形结构
- **创建壳体/实体**: 控制输出类型

### 检查修复

**用途**: 检测并修复9种常见几何问题

**支持检查的问题类型**:

| 类型 | 检测标准 | 修复方法 |
|------|---------|---------|
| 狭小面 | 长宽比过大 | 合并/删除 |
| 微小面 | 面积过小 | 删除并填补 |
| 尖刺 | 几何尖刺特征 | 修剪 |
| 重复面 | 几何重复 | 删除重复 |
| 相交面 | 自相交/相交 | 分割/修剪 |
| 冗余点 | 度为2的顶点 | 合并边 |
| 冗余边 | 共线可合并 | 合并边 |
| 微小边 | 长度过小 | 删除/合并 |
| 间隙 | 边之间的间隙 | 缝合/补面 |

**使用步骤**:
1. 几何 → 检查修复
2. 执行检查
3. 在结果列表中查看问题
4. 使用筛选器过滤显示 (类型/严重程度/状态)
5. 选择要修复的问题
6. 点击"修复选中"或"一键修复全部"

**修复状态**:
- 未修复 (白色)
- 修复中 (黄色)
- 已修复 (绿色)
- 修复失败 (红色)
- 已忽略 (灰色)

### N边曲面补面

**用途**: 自动填充N边形孔洞

**使用步骤**:
1. 几何 → N边补面
2. 选择边界边 (至少3条边组成闭合环)
3. 选择补面方法
4. 设置参数
5. 点击"补面"

**补面方法**:

| 方法 | 适用场景 | 特点 |
|------|---------|------|
| 自动 | 一般场景 | 根据边界自动选择最佳方法 |
| 曲面拟合 | 复杂边界 | 质量最高，计算较慢 |
| Coons曲面 | 4边边界 | 经典方法，效率高 |
| B-Spline | N边边界 | 通用性强 |
| 直纹面 | 线框边界 | 简单快速 |

**参数说明**:
- **次数**: B样条曲面的次数
- **分段数**: U/V方向的分段数量
- **公差**: 拟合公差
- **保持切向连续**: 与相邻面保持G1连续

### 删除面

**用途**: 删除选定的面并自动填补产生的孔洞

**使用步骤**:
1. 几何 → 删除面
2. 在视口或模型树中选择要删除的面
3. 选择填补策略
4. 点击"删除"或"智能删除"

**填补策略**:
- **自动**: 根据孔洞特征自动选择
- **平面填充**: 使用平面填充
- **最佳拟合**: 使用最佳拟合曲面
- **切向连续**: 与相邻面保持切向连续
- **补面引擎**: 使用N边补面引擎
- **保持开放**: 不填补，保持开放

### 抽梁

**用途**: 从实体或壳体模型中提取梁结构

**使用步骤**:
1. 几何 → 抽梁
2. 设置提取参数
3. 点击"提取梁"
4. 查看结果表格
5. 可选：导出梁信息

**参数说明**:
- **最小长度**: 忽略短于此值的梁段
- **最小截面面积**: 忽略小截面
- **最大曲率**: 曲率阈值
- **使用骨架提取**: 使用中心线方法
- **检测分支**: 识别T型/Y型连接
- **允许变截面**: 支持锥形梁

**输出信息**:
- 梁ID
- 总长度
- 段数
- 截面类型 (圆形/矩形/I型等)
- 是否变截面
- 体积

| 格式 | 扩展名 | 说明 |

| STEP | .stp | 标准STEP格式 |
| Parasolid XT | .x_t | Parasolid原生格式 |

## 常见问题

### Q: 为什么检查修复功能显示"未修复"状态?

A: 检查修复功能需要先执行"检查"，然后在结果列表中选择问题，最后点击"修复选中"或"一键修复"。

### Q: 缝合时出现"间隙过大"错误怎么办?

A: 可以尝试以下方法：
1. 增大缝合公差
2. 使用"智能缝合"自动调整公差
3. 先使用"检查修复"修复间隙问题
4. 手动使用"N边补面"填充大间隙

### Q: 如何识别重复面?

A: 在"检查修复"对话框中：
1. 点击"检查"
2. 在类型筛选中选择"重复面"
3. 系统会列出所有重复面
4. 选择并修复

### Q: 抽梁功能无法识别梁结构?

A: 检查以下设置：
1. 降低"最小长度"阈值
2. 降低"最小截面面积"阈值
3. 启用"使用特征识别"
4. 确保模型具有明显的梁特征

### Q: 如何保持与相邻面的连续性?

A: 在"N边补面"对话框中：
1. 勾选"保持切向连续"
2. 使用"曲面拟合"方法
3. 确保边界边正确选择了相邻面的信息
